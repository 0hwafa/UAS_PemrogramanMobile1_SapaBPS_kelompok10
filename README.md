# SAPA BPS (Sistem Aplikasi Pendataan Tamu BPS)

SAPA BPS adalah aplikasi *mobile* berbasis Android Native (Kotlin) yang dirancang untuk memfasilitasi dan mengelola pendataan tamu di lingkungan Badan Pusat Statistik (BPS) Kota Bandung. Aplikasi ini dilengkapi dengan sistem autentikasi (Login/Register), manajemen riwayat antrian secara *real-time* dan generate QR, serta fitur pemindaian kode QR khusus untuk peran Admin.

Proyek ini dikembangkan sebagai syarat pemenuhan tugas Ujian Akhir Semester (UAS) mata kuliah Pemrograman Mobile 1, Kelas TIF RP 24C CID.
